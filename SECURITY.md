# Security Policy

## Supported versions

Security fixes are provided for the latest published minor line of:

- `@jilimb0/tgwrapper`
- `@jilimb0/tgwrapper-adapter-redis`
- `@jilimb0/tgwrapper-observability`

## Reporting a vulnerability

Please report vulnerabilities privately and do not open a public issue.

Preferred channel:

- GitHub Security Advisory (private report)

If GitHub advisories are unavailable, contact maintainers directly through repository contact channels and include:

- affected package and version
- impact summary
- reproduction or proof of concept
- suggested mitigation (if known)

## Disclosure process

- We acknowledge reports as quickly as possible.
- We validate impact and scope.
- We prepare a fix and release.
- We publish a coordinated advisory with remediation guidance.
