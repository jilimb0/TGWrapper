# TGWrapper Comparison

## TGWrapper vs Telegraf
- TGWrapper emphasizes serverless-first runtime portability and explicit release/compatibility gates.
- Telegraf has mature ecosystem breadth and middleware familiarity.

## TGWrapper vs grammY
- TGWrapper emphasizes production checklists, schema/baseline gating, and operational release controls.
- grammY has a large plugin ecosystem and broad community adoption.

## TGWrapper vs GramIO
- TGWrapper emphasizes typed contracts + operational rigor (release gates, compatibility verification, observability path).
- GramIO focuses on developer ergonomics and modern toolkit workflows.

## Choosing TGWrapper
Choose TGWrapper when you want production-focused patterns out of the box: typed API contracts, release discipline, and serverless portability.
